## 本程序启动参数

参考: [Java 17](https://docs.oracle.com/en/java/javase/17/docs/specs/man/java.html)

```text
# ------------------------------------------------------------------------------------
# 杂项
# ------------------------------------------------------------------------------------

# 打印所有虚拟机参数
-XX:+PrintCommandLineFlags

# 关闭查看所有虚拟机参数的默认值
-XX:-PrintFlagsInitial

# 查看所有虚拟机参数的本虚拟机实例的实际值
-XX:+PrintFlagsFinal

# ------------------------------------------------------------------------------------
# 执行引擎
# ------------------------------------------------------------------------------------

# 服务器模式 使用更激进的JIT优化方案
-server

# 解释执行与JIT代码混合方式
-Xmixed

# ------------------------------------------------------------------------------------
# 内存 (堆)
# ------------------------------------------------------------------------------------

# 在堆空间内存溢出时生成dump文件
-XX:+HeapDumpOnOutOfMemoryError
-XX:HeapDumpPath=./Java-HeapDump.hprof

# 初始堆内存
-XX:InitialHeapSize=16g

# 最小/最大 堆内存
-XX:MinHeapSize=16g
-XX:MaxHeapSize=16g

# 堆内存中新生代和老年代的尺寸关系 默认值为2, 即1:2
-XX:NewRatio=2

# 伊甸园区与S0/S1的比例关系 默认值为8，即 8:1:1
-XX:SurvivorRatio=8

# 新生代对象经理过n此GC则被保存到老年代
-XX:MaxTenuringThreshold=15

# 是否在伊甸园区开辟线程专属内存空间 默认为开启
-XX:+UseTLAB

# ------------------------------------------------------------------------------------
# 内存 (栈)
# ------------------------------------------------------------------------------------

# JVM栈每个线程的栈占用尺寸
-XX:ThreadStackSize=1m

# ------------------------------------------------------------------------------------
# 内存 (元数据)
# ------------------------------------------------------------------------------------

# 元数据区大小一般不受限
-XX:MaxMetaspaceSize=16g
```
